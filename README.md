# DevCalculator

Developer calculator built with node-webkit, javascript, html and css.

# Build
  * `npm install`
  * `grunt build[:<target>]` where `<target>` is `all || [win,osx,linux32,linux64]`.
  Defaults to the current platform.

# dev

 * `npm install`
 * `grunt run`
